## 5. Analisis FODA del Mercado (Oportunidades para ProHo)

### Oportunidades

1. **Transparencia en precios** - La mayoria no publica precios. ProHo puede diferenciarse con pricing transparente y autoservicio.
2. **Modelo freemium/PLG** - Solo Kipo ofrece esto. Hay espacio para un modelo product-led growth mas sofisticado.
3. **IA como diferenciador real** - Solo Munily tiene IA. Hay oportunidad de integrar IA de forma mas profunda (prediccion de morosidad, automatizacion de comunicaciones, analisis de gastos).
4. **Integraciones abiertas** - Ningun competidor destaca por integraciones con terceros (bancos, ERP, contabilidad externa). Un ecosistema abierto seria diferenciador.
5. **UX moderna** - Muchas plataformas tienen interfaces anticuadas, especialmente las contables (Helisa, DecoPropiedad).
6. **Segmento desatendido** - Conjuntos pequenos/medianos (<100 unidades) que no justifican soluciones enterprise.
7. **Asambleas digitales** - Post-pandemia la demanda de asambleas virtuales se consolido; ConnextData es el unico especializado.

### Amenazas

1. **Munily** tiene backing de YC y esta escalando agresivamente con IA
2. **Kipo** con modelo gratuito puede capturar market share rapidamente
3. **Edifito** ya tiene presencia en 9 paises (ventaja si ProHo quiere internacionalizarse)
4. **Fragmentacion** del mercado - muchos jugadores pequenos compitiendo por el mismo segmento

### Posibles diferenciadores para ProHo

1. **Open API / Integraciones** - Conectar con sistemas contables existentes, bancos, pasarelas de pago
2. **IA generativa** para comunicaciones automaticas, resumenes de asambleas, analisis de morosidad
3. **Onboarding self-service** con pricing transparente
4. **Dashboard para juntas directivas** con KPIs en tiempo real
5. **Multi-tenancy para administradoras** que manejan multiples conjuntos
